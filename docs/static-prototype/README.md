# Virginia Dental Care Redesign Pitch Prototype

This is a static homepage redesign concept for `vadentalcare.com`, built as a pitch-ready prototype.

## What is included

- Full-bleed patient-care hero with booking and call CTAs.
- Clear service navigation for general, cosmetic, periodontal, implant, and emergency dentistry.
- First-visit flow that reduces patient uncertainty.
- Testimonial section using review language from the current site.
- Location, hours, phone, booking, and map section.
- Mobile sticky Call / Book action bar.

## Open locally

Open `index.html` directly in a browser, or serve the folder with any static server.

## Pitch angle

The current site has the right business facts, but the redesign makes the conversion path clearer: phone, booking, location, first-visit expectations, transparent pricing, and service categories all become easier to scan in the first few seconds.
